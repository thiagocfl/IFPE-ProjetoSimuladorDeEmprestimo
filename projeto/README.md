# Projeto Simulador de Empréstimo
***
Projeto desenvolvido em grupo, na linguagem _**Pascal**_ para a disciplina de _**Lógica da Programção 1**_.

O objetivo do projeto foi avaliar os conhecimentos dos assuntos abordados no 1º perído do curso técnico de informática.

Estruturas utilizadas:
* De repetição
    * while (enquanto) "a expressão for verdadeira"
* Condicional
    * if (se)
    * then (então)
    * else (senão)
